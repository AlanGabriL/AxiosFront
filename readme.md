### Tecnologia usada

- Visual Studio Code Editor

- Git e GitHub

- Insomnia _para testes_

- Node Js

- Express

- Cors

- Axios

### Comandos

- Clona o Projeto

  - ```cmd
      git clone https://github.com/jakeliny/node-api-discover.git
    ```

- Instala as dependencias

  - ```cmd
      npm i
    ```

- Executa o projeto

  - ```cmd
      nodemon .
    ```
